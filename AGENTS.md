## Required pre-finish gate

`npm run check` must pass before any change is considered done. It runs `react-router typegen && tsc && eslint . && prettier --check .`.

## Don'ts

- Don't silence the type checker. No `any`, `@ts-ignore`, or unjustified `eslint-disable` — fix the underlying issue.
- Don't edit files under `db/drizzle/` by hand. They're regenerated by `npm run db:generate`.
- Don't write your own `LoaderArgs` / `ActionArgs` types. Import the generated `Route` types from `./+types/<name>`.
- Don't import the `db` client into routes or worker handlers. Go through [db/repositories/](db/repositories/).
- Don't build UI from scratch when a primitive in [web/components/ui-kit/](web/components/ui-kit) already fits.
